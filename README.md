#### 申请 CI 或 webhook

mirai 可以为你提供一个 GitHub CI 或 GitHub webhook 以便创建第三方镜像库.   
你可以通过 PR 添加 CI 文件, 或者通过 issue 提交 webhook 申请.  
在申请时请 `@Him188`, 否则不会收到回复.

若你愿意, 你的镜像库链接也可以放在 README 这里.
